### Downloading all the playgrounds

除非另有说明，否则所有游乐场均与Swift 3兼容.

所有游乐场都可以在`playgrounds /`目录中作为子模块使用，要一次性下载所有游乐场，只需使用git clone --recursive https://github.com/uraimo/Awesome-Swift-Playgrounds克隆此存储库即可. git或以通常方式克隆存储库后执行git submodule update --init. 

必须手动下载以zip存档形式分发的Apple游乐场.
