#### High quality GIF

与ffmpeg /基于此 [article](http://blog.pkh.me/p/21-high-quality-gif-with-ffmpeg.html)

-生成调色板：

```bash
#!/bin/sh
start_time=30
duration=3
ffmpeg -y -ss $ start_time -t $ duration -i input.avi \
-vf fps = 10，scale = 320：-1：flags = lanczos，palettegenpalette.png
```
-使用调色板输出GIF：

```bash
#!/bin/sh
start_time=30
duration=3
ffmpeg -ss $ start_time -t $ duration -i input.avi -i Palette.png -filter_complex \
“ fps = 10，scale = 320：-1：flags = lanczos [x]; [x] [1：v] paletteuse” output.gif
```
