[Grakn](https://grakn.ai) 是一个数据库； 一种通过逻辑，推理和知识工程使复杂系统变得更智能的方法.

## IDE and editors support

- [`emacs-graql-mode`](https://github.com/creatorrr/emacs-graql-mode) -主要模式，为Graql提供语法高亮显示和基本完成.
- [`graql-language-vscode`](https://github.com/idealley/graql-language-vscode) -用于Visual Studio代码的Graql语言.
- [`JetBrains-Graql-Plugin`](https://github.com/BFergerson/JetBrains-Graql-Plugin) -JetBrains对Graql语言的支持.
- [`language-graql`](https://github.com/ShishkinDmitriy/language-graql) -Graql的Atom语法突出显示.


## Open source projects using Grakn

- [`Source++`](https://github.com/sourceplusplus/Assistant) -具有运行时感知和上下文感知功能的开源自动编程助手.
- [`Rasa`](https://github.com/RasaHQ/rasa) -开源机器学习框架，可自动执行基于文本和语音的对话.
- [`Grakn Mythos`](https://github.com/BFergerson/grakn-mythos) -基于Grakn的应用程序，用于创建和共享知识图.
