### Books for Algorithms

 |  ☆| 姓名| 描述

| ★★★ | [Algorithms](https://www.amazon.com/Algorithms-4th-Robert-Sedgewick/dp/032157351X)由Robert Sedgewick和Kevin Wayne撰写| 这本书被整齐地分类，加上详尽的解释和奇妙的插图. 在某些IOI训练营中，它被用作教科书.  |
|     | [Algorithms and Data Structures in Action](https://www.manning.com/books/algorithms-and-data-structures-in-action) ，由Marcello La Rocca | 本书提供了一种不同的算法方法，以更实际的角度平衡了理论，每一章的重点是如何将算法应用于日常工作或竞争性编程中的实际用例. 它还提出了经典，高级和新算法的融合.  |

## Sites for Practice
&gt;良好的在线裁判系统/竞赛平台可以练习.

 |  ☆| 姓名| 描述
| --- | --- | --- |
| ★★★ | [Codeforces](http://codeforces.com/)  | 如果不是的话，Codeforces是目前最受欢迎的竞赛平台之一. 目前由萨拉托夫州立大学（Saratov State University）维护，定期举办比赛，并提供无数令人敬畏的原创问题. 此外，每场比赛都会（通常）提供作者自己写的即时有用的教程.  Codeforces还拥有一个强大而富有吸引力的社区. 总而言之，在这里确实可以学习和改进.  |
| ★★★ | [Topcoder](https://www.topcoder.com/)  |  Topcoder自2001年以来一直存在.历史悠久，在技术竞赛中，Topcoder被认为是最负盛名的组织之一. 数以百计的SRM产生了很多问题. 这里的问题通常比其他问题更具挑战性，因此Topcoder吸引了许多精英程序员. 一年一度的Topcoder Open（TCO）也是一个受到广泛讨论的活动.  |
| ★★★ | [Google Code Jam](https://code.google.com/codejam/)  |  Google Code Jam当然是最受尊敬的编程竞赛之一. 比赛包括独特的编程挑战，必须在固定时间内解决. 竞争对手可以使用任何编程语言和开发环境来获取解决方案.  |
